# Exercicis — Lliçó 4: Spacing

## Exercici 1: Jerarquia amb espaiat

Crea un HTML que mostri:

- Un títol principal amb `mb-6` (separació inferior)
- Un subtítol amb `mb-4`
- Un paràgraf amb `mb-8`
- Un botó amb `px-8 py-4` (padding generós)

Tot dins d'un contenidor `max-w-lg mx-auto p-6`.

---

## Exercici 2: Centrat horitzontal

Crea un quadre de 200px d'ample (`w-48`) amb fons gris clar. Centra'l horitzontalment dins del `<body>`.

Quina classe fa el centrat?

---

## Exercici 3: Separació entre elements

Sense fer servir `m-` o `ml-` als elements individuals, crea una llista vertical de 4 elements (`<div>`) separats 8px entre si.

**Pista:** fes servir un contenidor pare amb `space-y-2`.

---

## Exercici 4: Targeta amb espaiat

Crea una targeta que tingui:

- Padding interior de `p-6`
- Títol amb `mb-2`
- Text descriptiu amb `mb-4`
- Botó a baix a la dreta amb `text-right`
- Separació entre el text i el botó amb `mt-4`

---

## Exercici 5 (extra): Perfil de xarxa social

Recrea aquest disseny usant només espaiats:

```
┌──────────────────────────┐
│                          │
│   [Avatar]  Nom Usuari   │
│             @username     │
│                          │
│   Bio: Aprenent de CSS   │
│                          │
│   [Segueix] [Missatge]   │
│                          │
└──────────────────────────┘
```

Els botons han d'estar separats entre si (`space-x-3`) i del contingut superior (`mt-6`).
