Configuració (tailwind.config.js)

Personalització de colors i fonts al fitxer de configuració

module.exports = {
  content: ['./src/**/*.{html,js}'],
  theme: {
    extend: {
      colors: {
        brand: { 500: '#6c63ff', 700: '#5a52d5' },
      },
      fontFamily: {
        sans: ['Inter', 'sans-serif'],
      },
    },
  },
};

Nota: Aquest és codi JavaScript de configuració, no HTML renderitzat. Mostrat aquí amb finalitats il·lustratives.
