flex-1 (créixer per igual)

flex-1
flex-1
<div class="flex border-2 border-gray-400 p-2 gap-2">
  <div class="flex-1 child bg-red-300">flex-1</div>
  <div class="flex-1 child bg-blue-300">flex-1</div>
</div>

Cada fill ocupa la meitat de l'espai disponible gràcies a flex-1.

Variants de flex

flex-1
flex-auto
flex-initial
flex-none
<div class="flex border-2 border-gray-400 p-2 gap-2">
  <div class="flex-1 child bg-red-200">flex-1</div>
  <div class="flex-auto child bg-green-200">flex-auto</div>
  <div class="flex-initial child bg-blue-200">flex-initial</div>
  <div class="flex-none child bg-yellow-200">flex-none</div>
</div>

flex-1: base 0, creix. flex-auto: base auto, creix. flex-initial: base auto, no creix. flex-none: mida fixa.

Order (ordre visual)

1r HTML · 3r visual
2n HTML · 1r visual
3r HTML · 2n visual
<div class="flex border-2 border-gray-400 p-2 gap-2">
  <div class="order-3 child bg-red-300">1r HTML · 3r visual</div>
  <div class="order-1 child bg-green-300">2n HTML · 1r visual</div>
  <div class="order-2 child bg-blue-300">3r HTML · 2n visual</div>
</div>

L'ordre visual es reordena amb order-1, order-2, order-3.

Self alignment

self-start
self-center
self-end
self-stretch
<div class="flex items-start h-40 border-2 border-gray-400 p-2 gap-2">
  <div class="self-start child bg-red-200">self-start</div>
  <div class="self-center child bg-green-200">self-center</div>
  <div class="self-end child bg-blue-200">self-end</div>
  <div class="self-stretch child bg-yellow-200">self-stretch</div>
</div>

Cada fill pot sobrescriure l'align-items del pare.

Grow i shrink individuals

grow
grow-0
shrink
shrink-0 (w-32)
<div class="flex border-2 border-gray-400 p-2 gap-2">
  <div class="grow child bg-red-200">grow</div>
  <div class="grow-0 child bg-green-200">grow-0</div>
  <div class="shrink child bg-blue-200">shrink</div>
  <div class="shrink-0 child bg-yellow-200 w-32">shrink-0 (w-32)</div>
</div>

grow creix si hi ha espai. grow-0 no creix. shrink es pot encongir. shrink-0 no s'encongeix.

Targetes responsives amb flex-wrap

Targeta 1

S'adapta a l'ample disponible gràcies a flex-1 i min-w-[250px].

Targeta 2

Aquesta targeta té el mateix comportament responsive.

Targeta 3

Quan no hi cap, baixa a la següent línia.

<div class="flex flex-wrap gap-4 border-2 border-gray-400 p-4">
  <div class="flex-1 min-w-[250px] bg-white p-4 shadow rounded">
    <h3 class="font-bold text-lg">Targeta 1</h3>
    <p class="text-gray-600">S'adapta a l'ample disponible gràcies a <code>flex-1</code> i <code>min-w-[250px]</code>.</p>
  </div>
